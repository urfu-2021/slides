Домашняя работа "Форум"

Домашние работы на курсе

  • Работа в командах по 3-4 человека
  • Один проект на весь семестр, несколько этапов, в итоге получится мессенджер
  • Нужно реализовывать фронтовую часть, бэкенд (API) – уже реализован
  • Будут практики, чтобы поработать очно и проконсультироваться
  • Когда задание готово – его проверяет ментор

Первый этап. Формум

  • Работаем в общем репозитории https://github.com/urfu-2021/forum-team-n, n – номер команды
  • Делаем простой форум, куда могут отправлять сообщения анонимные пользователи
  • Используем для верстки и логики HTML, CSS и JS, без использования библиотек
